# Projects

## Spatial Risk Forecasting and Route Optimization

**UC Berkeley STAT 222 Capstone Project**

Built a spatial risk forecasting and route-scoring framework for San Francisco using historical reported crime data. The project divided downtown San Francisco into 0.1-mile grid cells, modeled grid-level crime counts using zero-inflated negative binomial regression, and ranked alternative walking routes by estimated cumulative risk.

**Methods:** zero-inflated negative binomial regression, spatial aggregation, count modeling, route scoring, R, Google Maps API.
