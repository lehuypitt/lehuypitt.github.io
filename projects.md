# Projects

## Spatial Risk Forecasting and Route Optimization

**UC Berkeley STAT 222 Capstone Project**

Built a spatial risk forecasting and route-scoring framework for San Francisco using historical reported crime data. The project divided downtown San Francisco into 0.1-mile grid cells, modeled grid-level crime counts using zero-inflated negative binomial regression, and ranked alternative walking routes by estimated cumulative risk.

**Methods:** zero-inflated negative binomial regression, spatial aggregation, count modeling, route scoring, R, Google Maps API.

[GitHub Repository](https://github.com/lehuypitt/spatial-risk-route-optimization)

## Adaptive Rejection Sampling R Package

**UC Berkeley Statistical Computing Project**

Built an R package implementing adaptive rejection sampling for one-dimensional log-concave density functions. The package includes a user-facing `ars()` function, numerical derivative calculations, initialization routines, upper/lower hull updates, and unit tests.

**Methods:** Monte Carlo simulation, rejection sampling, log-concave densities, numerical optimization, R package development.

[GitHub Repository](https://github.com/lehuypitt/adaptive-rejection-sampling-r)
